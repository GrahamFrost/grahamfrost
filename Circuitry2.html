Circuitry - project 2 | Reflection

by Graham /

This circuit is a 555 timer possibly the most used chip / circuit in the technology industry. It can do many things such as; Lamp flashers, pulse generation logic clocks, tone generation, security alarms, pulse position modulation and many more. In this example I have the timer set up to be a clock with a pulse of 1 pulse per second. What I learned in this lab was to be able to create this and I feel that knowing how to put togeather one of the most used circuits in the industry is a very good skill to have.
