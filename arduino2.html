Arduino - FinalProject Temperature Sensor | Reflection

by Graham

During our final project, one of the tasks we decided for ourselves would be to use an arduino to display the temperature of the CPU within the computer. In order to do this we had to write code and use some from resources we found online and from our teachers. This worked well however in order for the arduino to display using a touch screen we had to encorporate another arduino to do the calculations on and then send them to the ardunio with the touch shield attached as the arduinos can only output through 1 "COM" port. We had to use two, one to send the information to the arduino and one to send the code to the touch screen. In order to work around this I had to solder a piece of wire onto the highly sensitive touch shield. I found that this was a practical use of my skills I had learned, proving after a year from the soldering course that I had retained the information I had learned.
